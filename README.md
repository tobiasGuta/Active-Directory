# Active-Directory

This repository contains tools developed for **Network Enumeration and System Analysis**. 

These scripts are designed to help security professionals and students understand:
1.  **Offensive Security:** How attackers enumerate networks and identify vulnerabilities.
2.  **Defensive Security:** What logs are generated by these activities and how to detect them.

**Do not use these tools on unauthorized systems.** Accessing a computer network without authorization is a crime. The author is not responsible for any misuse of this code.
